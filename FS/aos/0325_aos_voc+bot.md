## 内容结构

### AWS GenAI 技术栈 (Slide 1-6)

- **三层架构：**
  - 底层：基础设施 — 自研芯片 Trainium/Inferentia、EC2、SageMaker、UltraClusters、Nitro、EFA、Neuron
  - 中间层：Amazon Bedrock — FM 模型平台（Amazon Nova、Claude、DeepSeek、Llama、Stable Diffusion），含 Guardrails、Agents、Customization
  - 上层：应用 — Amazon Q、QuickSight、Connect
- **亚马逊 GenAI 战略：**
  - 自研加速计算芯片（Trn2 性价比比 P5e/P5en 高 30-40%）
  - Anthropic 选择 AWS 作为主要云供应商
  - Project Rainier — 全球最大 AI 计算集群（数十万 Trainium 芯片）
  - 飞轮效应：芯片 → 模型生态 → 应用 → 数据反馈
